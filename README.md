# me-rewards-cdn

## Install

```html
<script src="https://cdn.jsdelivr.net/gh/Me-Protocol/mepaas-reward-cdn@main/cdn.js"></script>
```

```html

  window.setCustomerData({
    email: "address@email.com",

    firstName: "Name",
    lastName: "Last Name",

```
